# Teleconverter

Converts telecom metrics to other metrics.
  
### Requirements

* Python >= 3.10
* Tkinter >= 8.6

Install them on Debian based Linux distros with:
```bash
apt install python3 python3-tk
```
On Fedora and RHEL based distros:
```
dnf install python3 python3-tkinter
```
